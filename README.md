# MYSQL数据迁移工具

一款简单易用的MySQL数据迁移工具，方便非专业技术人员对数据进行`备份`/`恢复`。

**UI**：基于 **[AntdUI](https://gitee.com/antdui/AntdUI)**

**技术框架**：.NET8 WinForms

**测试兼容**：`Win10`/`Win11`+`MySQL 8.0`
